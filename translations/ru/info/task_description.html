В программировании дерево - абстрактный тип представления информации в иерархической структруре. Все элементы дерева называются узлами (или нодами). Линии, соединяющие узлы дерева, называют ветвями, а узлы без потомков называют листьями или конечными узлами. Узел, не имеющий родителя, называют корнем или корневым узлом.

В этой миссии в качестве входных данных даются дерево данных произвольной структуры и определённое целевое значение. Требуется подсчитать количество листьев данного дерева, значения которых равны целевому значению. Узлы могут быть представлены любым из типом данных из этого списка (или их комбинацией): int, str, list, tuple, dict.
Чтобы лучше понимать, о чём идёт речь, посмотри на иллюстрации примеров ниже. Если при решении задачи возникнут проблемы, задумайся над применением итеративного подхода разработки и посмотри это видео, демонстрирующее, как этот подход может применяться при решение задачи.

Входные данные: Два аргумента:

  • первый - произвольная структура данных (tree)
  • второй - целевое значение (target), которое нужно искать в листьях дерева, может иметь любой тип из списка int, str, list, tuple, dict

Выходные данные: Количество листьев дерева, соответствующих целевому значению (целое число).

Предварительные условия:

  • n ∈ [1; 1000] - количество узлов дерева
  • каждый узел дерева может быть представлен любым типом данных (или их комбинацией) из списка int, str, list, tuple, dict
