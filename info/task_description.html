In computer science, a tree is an abstract data type that simulates a hierarchical tree structure. All tree elements are called nodes. The lines connecting elements (or nodes) are called branches and nodes without children are called leaves or end-nodes. The topmost node in a tree is called the root node.

In this mission you are given an arbitrary tree and a target value, your task is to calculate how many leaves of the tree are to the target value. The nodes of the tree might be represented by any of these data types and/or their combinations: int, str, list, tuple, dict.
For better understanding have a look at the illustrations below. If encounter difficulties, try to watch this video exposing the use of the incremental development approach.

Input: Two arguments:

  • the first is an arbitrary tree data structure
  • the second is a target value, any type of [int, str, list, tuple, dict]

Output: The number of the leaves are equal to the given target value, integer.
